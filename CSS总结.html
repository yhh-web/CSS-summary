CSS总结

1、CSS说明：

CSS是级联样式表（Cascading Style Sheets）的缩写。HTML 用于撰写页面的内容，而 CSS 将决定这些内容该如何在屏幕上呈现,网页的内容是由 HTML的元素构建的，这些元素如何呈现，涉及许多方面，如整个页面的布局，元素的位置、距离、颜色、大小、是否显示、是否浮动、透明度等等。 简单来说CSS就是对HTML中的内容进行布局。

CSS的构成：h1{属性：值；属性：值；}

1、选择器是您需要改变样式的对象

2、每条声明由一个属性和一个值组成。（无论是一条或多条声明，都需要用{}包裹，且声明用;分割）

3、属性是您希望设置的样式属性,每个属性有一个值。属性和值被冒号分开。

选择器：包括元素选择器、id选择器和class选择器

id选择器：前有'#'号，由于元素id值必须唯一，所以id选择器适用范围只有一个元素。

class选择器：前有'.'号，页面上有class的就让它显示什么色，class值可多个可重复，所以class应用普遍。

CSS如何生效：外部样式表，内部样式表，内联样式

外部样式表：重新创建一个文件夹存放样式表，这样使样式表与 HTML 内容分开，结构清晰。同时其它页面也可使用，达到复用的目的。

内部样式表：将样式放在HTML文件中，在head元素中引入style元素，放入样式，只在页面样式规则少时采用。 内联样式：完全将内容和样式合在一起，不灵活，哪个样式定义离元素近哪个就生效。

2、CSS模型：

颜色，尺寸，对齐

颜色：采用颜色名称或使用颜色RGB16进制值来设定前景或背景

尺寸：用height和width设定高度和长度，像素px，百分比%

对齐：对于元素中的文本，我们可以简单的设置text-align属性为left, center, right即可

盒子模型由内容，内边距，边框，外边距构成

Content 盒子的内容，如文本、图片等

Padding 填充，也叫内边距，即内容和边框之间的区域

Border 边框，默认不显示

Margin 外边距，边框以外与其它元素的区域

宽度=左外边距 + 左边框宽度 + 左内边距 + 内容宽度 + 右内边距 + 右边框宽度 + 右外边距，width只设置了内容的宽度

边框与边距都有四个方向

大概形式为X-Y，X为盒子模型中的四个内容，Y为上下左右

3、CSS相关操作：

定位用属性position，包括静态、相对、固定、绝对这些值,
设置了元素的position属性后，我们才能使用top, bottom, left, 
right属性，否则定位无效。表达形式为position：X，X为属性值

static:此为元素的默认定位方式

relative:对于静态位置进行偏移

fixed：元素固定不动，此时元素固定的位置仍由top, bottom, left, right属性确定，但相对的是浏览器屏幕可见区域

absolute:使元素相对于其最近设置了定位属性（非static）的父元素进行偏移。 如果该元素的所有父元素都没有设置定位属性，那么就相对于这个父元素

当元素溢出指定区域时，用overflow来处理溢出
部分包括溢出属性包含visible,hidden,scroll,auto几个值

visible 默认值，溢出部分不被裁剪，在区域外面显示

hidden 裁剪溢出部分且不可见

scroll 裁剪溢出部分，但提供上下和左右滚动条供显示

auto 裁剪溢出部分，视情况提供滚动条

设置float属性让某元素水平方向上向左或右进行移动，其周围的元素也会重新排列。

4、组合器及伪元素

除了元素、id和class外，还有后代选择器和子选择器

后代选择器：以空格作为分隔，如：.haha p 代表在div元素内有.haha这种类的所有元素 （如：

.haha p {
background-color: yellow;
}
<div class="haha"></div>

.haha p只影响在div内的内容，div外的不影响

子选择器： 也称为直接后代选择器，以>作为分隔，如：.haha > p 代表在有.haha类的元素内的直接p元素。(如

<html>
<head>
<style>
.haha > p {
  background-color: yellow;
}
</style>
</head>
<body>
<div class="haha">
<p>Paragraph 1 in the div .haha.</p>
<p>Paragraph 2 in the div .haha.</p>
<span>
    <p>Paragraph 3 in the div .haha - it is descendant but not immediate child.</p>
</span> <!-- not Child but Descendant -->
</div>
<p>Paragraph 4. Not in a div .haha.</p>
<p>Paragraph 5. Not in a div .haha.</p>
</body>
</html>

虽然段落3在.haha类中，但它的直接父元素是span，不是.haha的直接后代，所以不能选择。只有段落1、2有黄色背景。

伪类或伪元素用于定义元素的某种特定的状态或位置等

语法为：

/* 选择器后使用 : 号，再跟上某个伪类/伪元素 */
selector:pseudo-class/pseudo-element {
 property:value;
}
